# Skin Face Mapping – 3D models

Place minimalistic face models here for the interactive face mapping feature.

**Si no hay archivos .glb:** la app muestra una **esfera de ejemplo** (no un rostro) para que puedas usar el mapa y los marcadores. Añade los archivos siguientes para ver un rostro real.

## Required files

- **face-female.glb** – Female face model (glTF/GLB format)
- **face-male.glb** – Male face model (glTF/GLB format)

## Guidelines

- Prefer **low-poly** or **stylized** faces for performance and a clean, editorial look.
- Avoid hyperrealistic or “uncanny valley” assets.
- Single mesh or named submeshes help with region labeling (e.g. cheek, forehead).
- If the model is not found, the app will show a placeholder so you can still test markers and UI.

## Free resources (examples)

- [Sketchfab](https://sketchfab.com) – search for “stylized face” or “low poly head”, filter by downloadable.
- [Quaternius](https://quaternius.com) – free low-poly characters and body parts.

Export as **GLB** and name the files as above.
